

# pwncli debug 命令
`debug`是我开发的第一个命令，也是我使用次数最多的一个命令，其主要用于本地调试。`debug`命令支持的功能有：
- 配合`gdb`调试程序和`exp`脚本
- 非常方便下断点和执行`gdb`命令
- `hook`指定函数，可绕过反调试
- 设置`env`(LD_PRELOAD)，替换程序链接的`so`
- 在`main`函数前停住，以命中断点
